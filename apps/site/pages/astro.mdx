# Astro

`react-tweet` works great with Astro using the React integration.

## Installation

Follow the [installation docs in the Introduction](/#installation).

Then add the React integration to your Astro app:

```bash copy
pnpm add @astrojs/react
```

Add it to your config:

```ts copy
// astro.config.mjs
import { defineConfig } from 'astro/config'
import react from '@astrojs/react'

export default defineConfig({
  integrations: [react()]
})
```

## Usage

Create a React component that renders a Tweet and use it from an Astro page:

```tsx copy
// src/react/TweetApp.tsx
import { Tweet } from 'react-tweet'

export default function TweetApp() {
  return <Tweet id="1628832338187636740" />
}
```

```astro copy
---
import TweetApp from '../react/TweetApp'
---

<main class="react-tweet-theme">
  <TweetApp client:load />
  <!-- You can use client:idle or client:visible as well -->
</main>
```

You can learn more about `Tweet` in the [Twitter theme docs](/twitter-theme).

## Running the example

Clone the [`react-tweet`](https://github.com/vercel/react-tweet) repository and then run:

```bash copy
pnpm install && pnpm dev --filter=astro-app...
```

The source for the example is at
[`apps/astro-app`](https://github.com/vercel/react-tweet/tree/main/apps/astro-app).
